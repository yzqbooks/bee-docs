新手指导​

如果您是首次了解 beego，我们建议您先阅读 快速入门。

其它资源​

如果您无法找到相关问题的满意答案，可以在 邮件列表 (beego-framework@googlegroups.com) 中搜索。

寻求帮助​

如果您遇到无法解决的问题，可以发送邮件到 beego-framework@googlegroups.com 提问。
